# 文档读取（.docx / .pdf / .xlsx / .pptx）

read_file 只能读纯文本。遇到 Office 文档或 PDF 时，用 `run_command` 执行 Python 一行命令提取文本。

---

## ⚠️ 核心原则（必须遵守）

1. **脚本必须 print()**：run_command 返回 stdout，print 了就拿到内容，不需要写临时文件
2. **不要写临时文件再读取**：那是多余的步骤。print → stdout → run_command 结果，一步到位
3. **提取内容是手段，不是目标**：用户说"读取/总结/分析"，拿到文本后还要完成用户的真正请求
4. **先建 todo**：① 提取文档内容 ② 完成用户要求（总结/分析/翻译等） ← 第②项才是目标

---

## 首次使用：安装依赖

```
run_command({ command: "pip install python-docx pymupdf openpyxl python-pptx" })
```

安装一次即可，后续无需重复。

---

## 标准流程（一步提取）

### .docx — 直接 run_command

```
run_command({ command: "python -c \"from docx import Document; doc=Document(r'文件路径.docx'); print('\\n'.join(p.text for p in doc.paragraphs))\"" })
```

内容较多或需要含表格时，用 write_file 写脚本再执行：

```python
from docx import Document

doc = Document(r"文件路径.docx")
for p in doc.paragraphs:
    print(p.text)
for table in doc.tables:
    for row in table.rows:
        print(" | ".join(cell.text for cell in row.cells))
```

```
write_file({ path: "临时路径/extract.py", content: "上面的代码", mode: "overwrite" })
run_command({ command: "python 临时路径/extract.py" })
```

→ run_command 的返回值就是文档全文，直接用于后续分析。

### .pdf

```
run_command({ command: "python -c \"import pymupdf; doc=pymupdf.open(r'文件路径.pdf'); [print(p.get_text()) for p in doc]\"" })
```

### .xlsx

```
run_command({ command: "python -c \"from openpyxl import load_workbook; wb=load_workbook(r'文件路径.xlsx',data_only=True); [print('\\t'.join(str(c) if c else '' for c in row)) for ws in wb for row in ws.iter_rows(values_only=True)]\"" })
```

### .pptx

```
run_command({ command: "python -c \"from pptx import Presentation; prs=Presentation(r'文件路径.pptx'); [print(s.text) for sl in prs.slides for s in sl.shapes if s.has_text_frame]\"" })
```

## ❌ 禁止的错误流程

```
写脚本（不 print，赋值给变量）→ 运行（无输出）→ 改脚本（写到临时 txt）→ 
运行 → 用 powershell 读 txt → 删除临时文件 → 汇报"清理完毕"
```

这个流程 **全部是多余步骤**，而且最后忘了回答用户的问题。

✅ **正确流程**：
```
run_command(python -c "...print(...)...")  →  stdout 就是文档内容  →  回答用户问题
```

## 批量处理（batch）

```
prompt_template: "步骤：
1. run_command 执行 python -c 提取 {{item}} 的文本内容（必须 print）
2. 如果 python -c 太长，用 write_file 写脚本 → run_command 执行
3. 从提取的内容中完成要求：[你需要的字段/总结/分析]
4. 用 write_file 输出结果"
```

## 常见问题

- **ModuleNotFoundError**: `run_command({ command: "pip install python-docx" })` 安装
- **.doc（旧格式）**: python-docx 不支持，需 `libreoffice --convert-to docx` 转换
- **.csv**: 纯文本，直接 read_file
- **内容截断**: run_command 最多返回 4000 字符，超长文档用脚本写法分页 print
