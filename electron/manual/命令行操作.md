# 命令行操作

`run_command` 底层走 `cmd.exe /c`，不能直接写 PowerShell 语法。  
需要 PowerShell 时，用 `powershell -NoProfile -Command "..."` 包一层。

---

## 磁盘空间

### 查看单个磁盘（如 D 盘）
```cmd
powershell -NoProfile -Command "Get-PSDrive D | Select-Object Name,@{N='总GB';E={[math]::Round(($_.Used+$_.Free)/1GB,1)}},@{N='可用GB';E={[math]::Round($_.Free/1GB,1)}}"
```
把 `D` 换成目标盘符即可。

### 查看所有磁盘
```cmd
powershell -NoProfile -Command "Get-PSDrive -PSProvider FileSystem | Select-Object Name,@{N='总GB';E={[math]::Round(($_.Used+$_.Free)/1GB,1)}},@{N='可用GB';E={[math]::Round($_.Free/1GB,1)}} | ft -Auto"
```

>  禁止用 `Get-CimInstance Win32_LogicalDisk` 查磁盘（WMI 有断连网络盘时会超时卡住）。  
> 禁止用 `dir C:\` 查磁盘空间（GBK 编码，输出乱码）。禁止用 `wmic`（Win11 已移除）。

## 内存

```cmd
powershell -NoProfile -Command "Get-CimInstance Win32_OperatingSystem | Select-Object @{N='总MB';E={[math]::Round($_.TotalVisibleMemorySize/1KB)}},@{N='可用MB';E={[math]::Round($_.FreePhysicalMemory/1KB)}} | ft -Auto"
```

## 进程

### 查看所有进程
```cmd
tasklist /fo table
```

### 按名称查找进程
```cmd
tasklist /fi "imagename eq chrome.exe"
```

### 结束进程
```cmd
taskkill /f /im 进程名.exe
```

### CPU 占用最高的前 10 个进程（实时）
```cmd
powershell -NoProfile -Command "Get-CimInstance Win32_PerfFormattedData_PerfProc_Process | Where-Object {$_.Name -ne '_Total' -and $_.Name -ne 'Idle'} | Sort-Object PercentProcessorTime -Desc | Select-Object -First 10 Name,IDProcess,PercentProcessorTime | ft -Auto"
```

### 当前总 CPU 占用率
```cmd
powershell -NoProfile -Command "Get-CimInstance Win32_PerfFormattedData_PerfOS_Processor | Where-Object Name -eq _Total | Select-Object -ExpandProperty PercentProcessorTime"
```

## 文件操作

### 列目录
```cmd
dir "D:\目录" /b
```
```cmd
powershell -NoProfile -Command "Get-ChildItem 'D:\目录' -Directory | Select-Object Name"
```

### 查找文件
```cmd
dir /s /b "%USERPROFILE%\Desktop\文件名.ext"
```

### 查看文本内容
```cmd
type "D:\path\to\file.txt"
```

### 获取桌面/文档/下载路径
```cmd
echo %USERPROFILE%\Desktop
```

###  禁止用命令行写文件
必须用 `write_file` 工具。`echo >` / `Set-Content` / `[IO.File]::WriteAllText` 均禁止。  
写文件前若需要路径，先用上方命令查出绝对路径，再传给 `write_file`。

## 注意事项

- **内层路径用单引号**：`-Command "Get-ChildItem 'D:\path'"`  cmd 不支持 `\"` 转义
- **WMI 命令（Get-CimInstance）可能较慢**，若超时可传 `timeoutMs: 60000`
- 禁止 `systeminfo | findstr "内存"`：GBK 输出，findstr UTF-8 匹配失败
- `dir /b` 只输出文件名无乱码；不加 `/b` 的 dir 表头是 GBK，查磁盘会乱码
