# 阿里校招查询工作流

## 概述
本工作流用于快速获取阿里巴巴官方发布的校园招聘信息，涵盖应届生招聘、实习生招聘两个大类的开放批次、岗位详情、投递入口等核心内容，适用于求职季学生、就业服务人员查询最新招聘信息，可有效规避第三方渠道的虚假、过期信息误导。

## 触发条件
何时应该使用此工作流：
- 用户明确提出查询阿里校招/实习生招聘相关需求
- 需要获取阿里最新校招官方岗位、投递链接等权威信息
- 前置条件：网络可正常访问阿里巴巴官方招聘域名

## 工具建议
**CRITICAL**: 必须明确列出每个工具的使用场景和调用方式：

- **打开阿里校招官方首页** → 使用 `browser_open`：
  ```typescript
  await tools.browser_open({ url: "https://talent-holding.alibaba.com/campus/home?lang=zh" });
  ```
  说明：首次访问阿里校招主站时调用，预期输出为浏览器成功加载阿里官方校招首页，显示当前开放的所有校招项目。

- **提取页面招聘相关信息** → 使用 `browser_read_page`：
  ```typescript
  await tools.browser_read_page({ selector: ".recruitment-project-list, .position-list-container" });
  ```
  说明：需要提取开放批次、毕业要求、岗位列表、更新时间等信息时调用，预期输出为结构化的文本招聘信息。

- **切换招聘批次/进入投递页面** → 使用 `browser_click`：
  ```typescript
  await tools.browser_click({ selector: "button:has-text('实习生招聘'), a:has-text('立即申请')" });
  ```
  说明：需要切换应届生/实习生招聘批次、查看岗位列表或进入投递入口时调用，预期输出为浏览器跳转至对应目标页面。

## 操作步骤
**每个步骤 = 2-5 分钟内可完成的原子操作**

### 步骤 1: 打开阿里校招官方首页
**目标**: 进入官方主站，获取当前所有开放校招项目基础信息
**执行**:
```typescript
await tools.browser_open({ url: "https://talent-holding.alibaba.com/campus/home?lang=zh" });
```
**预期结果**: 页面加载完成，显示当前开放的校招项目名称、对应毕业时间要求、官方咨询电话等基础信息。

### 步骤 2: 提取主站批次信息
**目标**: 结构化输出所有开放招聘批次的核心规则
**执行**:
```typescript
await tools.browser_read_page({ selector: ".project-card" });
```
**预期结果**: 获取批次名称、面向毕业时间范围、申请入口链接等信息，无遗漏核心规则。

### 步骤 3: 切换至目标招聘批次页面
**目标**: 进入指定的应届生/实习生招聘岗位列表页
**执行**:
```typescript
// 如需查询应届生招聘可将selector文本替换为「应届生招聘」
await tools.browser_click({ selector: "div[class*='project-card']:has-text('实习生招聘') .apply-btn" });
```
**预期结果**: 页面跳转至对应批次的岗位列表页，显示该批次下所有开放岗位的分类导航。

### 步骤 4: 提取岗位列表详情
**目标**: 输出目标批次下所有开放岗位的完整信息
**执行**:
```typescript
await tools.browser_read_page({ selector: ".position-item-container" });
```
**预期结果**: 结构化输出岗位名称、工作地点、更新时间、投递入口链接等核心信息。

## 常见陷阱
从实际对话中提取的错误模式和规避方法：
- ⚠️ **陷阱名称：非官方渠道信息失效**：不要使用搜索引擎返回的第三方招聘网站信息，容易出现过期、虚假岗位，必须直接访问`*.alibaba.com`域名的官方招聘页面。
- ⚠️ **陷阱名称：遗漏毕业时间限制**：不同校招批次面向的毕业时间范围不同，提取信息时必须明确标注毕业时间要求，避免给不符合条件的
